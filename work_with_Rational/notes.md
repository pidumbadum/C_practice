**union** — это специальный тип данных, где все поля делят ***одну и ту же ячейку памяти***.
```cpp
union {
    double d;              // Доступ как к числу с плавающей точкой
    unsigned long long b;  // Доступ к тем же байтам как к целому числу
} u;
```
Когда создаётся u, компилятор выделяет ровно 8 байт. И u.d, и u.b «смотрят» на эти же байты, просто интерпретируют их по-разному.

***unsigned long long*** — это базовый целочисленный тип данных в C++

```cpp 
unsigned long long exp = (u.b >> 52) & 0x7FF;
// 0x7FF = 0000...0000011111111111
unsigned long long mant = u.b & 0xFFFFFFFFFFFFFull;
// ull - говорит о том, что это именно лонг, лонг
```
